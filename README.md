## 게임 설명
  탱크 RPG는 2012년 Subset Games에서 출시된 인디게임 FTL : Faster Than Light에서 모티브를 얻은 텍스트 RPG 게임입니다.<br>
  플레이어는 전차를 조종하여 제한된 턴 이내에 보스까지 전진하여 격파해야 합니다.<br>
  한정된 연료, 포탄, 자원을 활용하여 살아남으세요.<br>

## 게임 플레이 방식
  숫자를 입력하여 진행합니다.<br>
  MainGame : 전차가 멈춰있는 상태로, 전진하거나 탐색하거나 급조 수리를 할 수 있습니다.<br>
            원하신다면, 지정된 키 이외의 키를 눌러 승무원들끼리 대화하는 장면도 볼 수 있습니다.<br>
            잘못된 값을 입력받았을 때에 대한 예외처리를 하던 도중에 활용해보면 좋을 것 같아서 넣어봤습니다.<br> 
  Encounter : 전차가 전진하고 있는 상태로, 확률적으로 적, 아군, 상점과 조우하게 됩니다.<br>
  BattleSystem : 전차가 적과 교전하는 상태로, 플레이어의 턴이 먼저 실행되고 그 뒤로 적들의 턴이 이어집니다.<br>
                 플레이어는 일반 공격, 포탄을 소비하는 특수 공격, 공격과 바꿔 1회성 버프를 얻는 회피기동, 전투에서 도주 등을 선택할 수 있습니다.<br>
